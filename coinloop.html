Project Overview

Coinloop is a mobile application which acts as an interface for hardware wallets like Trezor and Octowallet as well as a software wallet. This is the world's first mobile app which interfaces directly with the hardware wallet. Focusing on an intuitive interface, the app's main objective is to act as the perfect crypto wallet to be used anywhere on the go!

Portfolio Screen

The Portfolio Screen shows the user all the crypto assets they hold. They can tap on any one of them for details about all the transactions for that particular asset. The bottom navigation bar makes it easy for the user to use all the features of the app

Send and Receive

Send and receive crypto assets were the two of the most important actions the user could make using the app, simple to use but with several layers of security.

About the Project

This project was for Coin Loop Pty. Ltd. in Melbourne, Australia. There are two versions of the app — a React Native app as well as an Android Native app. I worked on both but this details the work only for the Android Native app.
